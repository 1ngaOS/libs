# 1ngaOS Libs

Reusable libraries and crates from **1nga Solutions**, used across 1ngaOS and related projects.

## Libraries

| Library | Description |
|--------|-------------|
| [**wangamail-rs**](./wangamail-rs/) | Send email via [Microsoft Graph API](https://learn.microsoft.com/en-us/graph/overview) using app registration (OAuth2 client credentials). Rust crate; part of the WangaMail family. |
| [**wangapayfast-rs**](./wangapayfast-rs/) | [PayFast](https://www.payfast.co.za/) ITN (Instant Transaction Notification) parsing and signature verification for Rust services. |

Each library has its own README, tests, and CI (format, lint, check, test; version bump and publish on push to `main` where configured).
